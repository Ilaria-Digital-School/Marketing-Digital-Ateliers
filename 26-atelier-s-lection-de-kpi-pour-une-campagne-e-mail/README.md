# Atelier : Sélection de KPI pour une campagne e-mail

**Rappel théorique** Un KPI (Key Performance Indicator) est un indicateur mesurable qui permet d'évaluer l'atteinte d'un objectif précis. Il se distingue d'une simple métrique par son caractère décisionnel : un KPI doit être pertinent par rapport à l'objectif visé, disponible dans les outils dont vous disposez, fiable dans sa collecte et actionnable, c'est-à-dire qu'il doit vous permettre de prendre une décision concrète si sa valeur est insatisfaisante. Dans le cadre d'une campagne e-mail, les KPI se répartissent généralement entre trois familles : l'acquisition (ex. : nombre d'abonnés nouveaux), l'engagement (ex. : taux d'ouverture, taux de clic) et la conversion (ex. : taux de conversion, chiffre d'affaires généré). L'erreur la plus fréquente des débutants est de choisir des KPI « par défaut » sans les relier à un objectif SMART précis — par exemple, suivre le taux d'ouverture alors que l'objectif réel est d'augmenter les ventes d'un produit spécifique. Mini-exemple : si l'objectif SMART est « Augmenter de 15 % les ventes du produit X via la newsletter dans les 30 prochains jours », le KPI principal pertinent est le taux de conversion (clics ayant abouti à un achat), et non simplement le taux d'ouverture, qui reste un KPI secondaire d'engagement. Gardez en tête ce modèle mental : Objectif SMART → Famille de KPI → KPI principal + KPI(s) secondaire(s) → Critères de qualité validés.
