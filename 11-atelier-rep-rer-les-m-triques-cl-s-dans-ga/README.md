# Atelier : repérer les métriques clés dans GA

**Rappel théorique** Google Analytics (GA) est un outil de mesure de la performance web qui collecte et organise les données de comportement des visiteurs d'un site. Avant de plonger dans l'interface, trois concepts fondamentaux sont à maîtriser : les **sessions** (une visite d'un utilisateur sur le site, quelle que soit la durée), les **utilisateurs** (le nombre de personnes distinctes ayant visité le site), et les **pages vues** (le nombre total de pages consultées). À ces trois piliers s'ajoutent des métriques de performance critiques déjà abordées dans les cours précédents : le **taux de rebond** (% de sessions où l'utilisateur quitte sans interagir), la **durée moyenne de session**, et le **taux de conversion** (% de visiteurs ayant accompli un objectif défini). Une erreur fréquente chez les débutants est de se focaliser uniquement sur le volume de trafic (nombre de sessions) sans croiser cette donnée avec la qualité de l'engagement. Un site peut recevoir 10 000 visites par mois avec un taux de rebond de 90 % : c'est un signal d'alarme, pas une victoire. Le bon réflexe est toujours de lire les métriques en contexte, en les reliant à une étape précise du funnel marketing. Mini-exemple : si votre page d'accueil enregistre un taux de rebond de 80 % et une durée moyenne de session de 15 secondes, cela indique que les visiteurs ne trouvent pas ce qu'ils cherchent, ou que votre proposition de valeur n'est pas suffisamment claire.
