# Atelier : Calculer baseline à partir de données fictives

**Rappel théorique** Avant de définir des objectifs SMART ou de choisir des KPI pertinents, il est indispensable de connaître son point de départ : c'est ce qu'on appelle la baseline (ligne de base). La baseline est la valeur de référence mesurée sur une période donnée, avant toute action ou optimisation. Elle répond à la question : « Où en suis-je aujourd'hui ? ». Sans baseline solide, tout objectif chiffré est arbitraire et tout écart de performance devient impossible à interpréter. La baseline se calcule généralement en faisant la moyenne d'une métrique sur une période représentative (souvent 30, 60 ou 90 jours). Elle doit s'appuyer sur des données fiables, disponibles et comparables dans le temps. Attention à trois erreurs fréquentes : utiliser une période atypique (soldes, lancement produit, panne technique), confondre la baseline avec le meilleur résultat historique, et négliger la saisonnalité. Exemple concret : si votre taux de conversion e-commerce est de 1,8 % en janvier, 2,1 % en février et 1,9 % en mars, votre baseline est (1,8 + 2,1 + 1,9) / 3 = 1,93 %. C'est ce chiffre, et non le pic de février, qui sert de référence pour fixer un objectif réaliste du type « atteindre 2,5 % d'ici 6 mois ». --- **Énoncé de l'atelier** Dans cet atelier, vous allez calculer les baselines de plusieurs KPI digitaux à partir d'un jeu de données fictives fourni ci-dessous.
