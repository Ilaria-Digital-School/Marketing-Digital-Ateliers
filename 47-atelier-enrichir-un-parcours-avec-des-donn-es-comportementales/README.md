# Atelier : enrichir un parcours avec des données comportementales

**Rappel théorique** Un Customer Journey Map (CJM) cartographie les étapes qu'un utilisateur traverse avant, pendant et après son interaction avec un produit ou service. Jusqu'ici, vous avez appris à identifier des personas, des touchpoints et à structurer un parcours basique. Mais un CJM statique, basé uniquement sur des suppositions ou des données déclaratives (ce que les gens disent faire), reste une hypothèse. Les données comportementales, elles, révèlent ce que les utilisateurs font réellement : temps passé sur une page, taux d'abandon, clics, rebonds, séquences de navigation. Enrichir un parcours avec ces données transforme un outil qualitatif en levier décisionnel concret. Le mental model clé ici : distinguer le parcours perçu (ce que vous pensez que l'utilisateur vit) du parcours réel (ce que les données montrent). L'erreur la plus fréquente des débutants est de construire un CJM uniquement à partir d'intuitions ou d'entretiens sans jamais le confronter à des données chiffrées. Un mini-exemple : vous cartographiez une étape "Découverte via les réseaux sociaux" dans votre parcours, mais Google Analytics révèle que 70 % du trafic provient de la recherche organique. Cette donnée doit remodeler votre carte et votre stratégie. La donnée comportementale n'invalide pas votre CJM, elle l'affine et le rend opérationnel. --- **Énoncé de l'atelier** Vous avez désormais les bases pour construire un parcours client structuré. L'objectif de cet atelier est de passer à l'étape supérieure : enrichir un CJM existant avec des données comportementales réelles ou simulées, afin de le rendre exploitable dans une logique de stratégie digitale.
