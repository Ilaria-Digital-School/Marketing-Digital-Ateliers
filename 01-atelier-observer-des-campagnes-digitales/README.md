# Atelier : observer des campagnes digitales

**Rappel théorique** Le marketing digital désigne l'ensemble des actions de communication et de promotion réalisées via des canaux numériques : moteurs de recherche, réseaux sociaux, e-mail, sites web, applications mobiles, etc. Une campagne digitale est une action marketing coordonnée, limitée dans le temps, visant un objectif précis (notoriété, acquisition, conversion, fidélisation). Elle repose sur plusieurs composantes clés : la cible (audience définie par des critères démographiques, comportementaux ou d'intérêt), le message (contenu créatif et copywriting), le canal (où la campagne est diffusée), le budget, et les indicateurs de performance (KPIs). Un modèle mental utile est le tunnel de conversion : Awareness (prise de conscience) → Consideration (évaluation) → Conversion (achat ou action) → Loyalty (fidélisation). Toute campagne s'inscrit dans l'une de ces phases. Erreur fréquente des débutants : confondre le canal (ex. Facebook Ads) avec l'objectif (ex. générer des leads). Le canal est un moyen, pas une fin. Mini-exemple : une marque de vêtements lance une publicité Instagram ciblant les 18-35 ans avec un code promo. L'objectif est la conversion. Le canal est Instagram. Le KPI principal est le taux de clics et le nombre de codes utilisés.
